Rock Paper Scissors

This project is a browser-based "Rock, Paper, Scissors" game with options for both manual play and automatic play. The game tracks and displays the player's win, loss, and tie records, storing the score in localStorage for persistence. Users can interact via buttons or keyboard shortcuts to make moves, while the automatic mode lets the computer play on behalf of the player every second. The UI dynamically updates to show the game result, player and computer moves, and the current score..
